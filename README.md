# Artakula

**Artakula** adalah aplikasi keuangan pribadi yang berfokus pada kesederhanaan, kecepatan, dan pengalaman penggunaan tanpa gangguan.

Aplikasi ini dibuat untuk membantu pengguna mengelola keuangan sehari-hari secara praktis, gratis, dan tanpa iklan.

---

## Tujuan Aplikasi

Menyediakan aplikasi pencatat keuangan pribadi yang:

* Mudah digunakan
* Ringan dan cepat
* Gratis selamanya
* Tanpa iklan
* Fokus pada kebutuhan inti pengelolaan keuangan

---

## Masalah yang Diselesaikan

Banyak aplikasi keuangan saat ini:

* Terlalu kompleks
* Dipenuhi fitur yang tidak diperlukan
* Mengandung iklan
* Membutuhkan langganan

**Artakula** hadir sebagai solusi sederhana untuk pencatatan dan pengelolaan keuangan pribadi.
